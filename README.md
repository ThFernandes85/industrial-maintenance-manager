# Industrial Maintenance Manager

🇺🇸 English below | 🇧🇷 Português abaixo

## 🇺🇸 Overview

Industrial Maintenance Manager is a simple system for registering and tracking maintenance work orders (OMs).

This project simulates how industrial maintenance activities can be managed digitally.

### Features
- Create maintenance work orders
- List all work orders
- Track status and creation date

### Tech Stack
- Python
- JSON storage
- Command line interface
